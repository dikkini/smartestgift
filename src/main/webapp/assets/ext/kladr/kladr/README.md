jQuery Kladr
================================================================================

Составляющие плагина по отдельности

* **jquery.kladr.core.min.js** - Реализует $.kladr. Позволяет обращаться к сервису [kladr-api.ru] [1].
* **jquery.kladr.plugin.min.js** - Реализует плагин $( '' ).kladr.

[1]: http://kladr-api.ru/        "КЛАДР API"
